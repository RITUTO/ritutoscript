コードの説明

log

文字を表示するためのコード
例 log text
コンソロールにtestと表示させられる
これは変数なども追加えます set message = hello log |message+world

set

変数を設定するためのコード例えば
set test = text
でtestにtextをいれれるまたこれには特別なワードがあるinputユザーに入力させれる
例
log あなたの名前は何ですか？
set name = input
log あなたの名前は+|name ですね
set num = 0
set num = ++
で1ずつ増やす
--で1ずつ減らす

if

どのプログラミング言語おなじみ？のif
if a = a {
log aはaです
}
if a ! b {
log aはbではありません
}
みたいな使い方
if 式 {
コード
}


常に指定されてる変数

|random
常に 1~100の間の乱数が指定されてる
|pressspace
スペースキーが押されたかどうか
if |pressspace = true {
log スペースキーが押されました
}
return
みたいな使い方をする

ボタン系

名前を変えるとき
set buttonname = buttontext
でボタンの名前がbuttontextになります 初期はbutton1 ボタン検知 if |buttonisclick ! false { log ボタンが押されました } return でボタンを押すとボタンが押されたと通知されます
